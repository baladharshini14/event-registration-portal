# Event Registration Portal

## Overview

The Event Registration Portal is a web-based application designed to simplify the process of organizing and registering for events. The system provides a centralized platform where organizers can create and manage events, while participants can browse available events and register online. The portal reduces manual work, improves accuracy, and enhances the overall event management experience.

## Problem Statement

Traditional event registration methods often rely on paper forms, spreadsheets, or manual record keeping. These approaches can lead to data duplication, registration errors, and difficulty in managing participant information. The Event Registration Portal addresses these challenges by automating the registration and event management process.

## Objectives

* Provide a simple and user-friendly event registration system.
* Allow participants to register for events online.
* Enable organizers to create, update, and manage events efficiently.
* Maintain participant records securely.
* Generate reports for event registrations and attendance.
* Improve communication between organizers and participants.

* Manage Users
* Manage Events

## System Modules

### 1. Authentication Module

Handles user registration, login, logout, and account management.

### 2. Event Management Module

Allows organizers and administrators to create, update, and manage events.

### 3. Registration Module

Manages participant registrations and event enrollment.

### 4. Notification Module

Sends registration confirmations and event-related updates.

### 5. Report Module

Generates reports related to registrations, participants, and event statistics.

## User Roles

### Organizer
